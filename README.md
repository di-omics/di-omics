### How I build

**Eval-first.** Plant known ground truth, run the method blind, score the recovery. Every repo here gates its claim on a number, not a screenshot, and shows the seam where a learned model earns its keep.

### Focus

Autonomous wet-lab execution · low-input epigenomics · PyLabRobot on Hamilton / Opentrons / Tecan · computer vision for lab video.
